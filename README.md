# ★ Sites favoritos

Passando de códigos Python para **HTML**! Esta atividade foi da minha agenda 12 de DSI, cujo objetivo foi desenvolver um site para armazenar os sites que eu mais uso.
Diferente das minhas outras atividades, duvido que esta tenha alguma atividade… a não ser que queiram saber um pouco mais sobre mim. 😄

Enfim, neste repositório estarão a pasta com os arquivos necessários para o site, mais imagens para comprovar o código usado e a execução funcional dele.
Tentei adicionar os comentários sobre o código dentro dele, usando o "#", como havia feito em outras atividades de Python, no entanto, o código do HTML não permite isso.
Portanto, apesar de não ter muito o que comentar, vou deixá-las aqui:
- Para distinguir o "cabeçalho" da página com o título dos sites, usei "h2" para cada nome, reservando o "h1" para o começo.
- Já que achei que era mais útil, usei o "_blank" para os links, assim sendo mais fácil abrir todos eles sem ter que voltar à página.
- E, para não deixar o trabalho muito "seco", adicionei um breve comentário sobre cada escolha, entre a imagem e o link.

Aproveite, lembre-se de colocar todos os arquivos numa mesma pasta!👋
